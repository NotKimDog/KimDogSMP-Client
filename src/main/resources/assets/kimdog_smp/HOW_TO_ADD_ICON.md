# How to Add Your Mod Icon

## Step 1: Create Your Icon
1. Create a **128x128** or **256x256** PNG image
2. Design it with your mod's theme (pickaxe, diamond, ores, etc.)
3. Name it `icon.png`

## Step 2: Place the Icon
Put your `icon.png` file in this directory:
```
src/main/resources/assets/kimdog_smp/icon.png
```

## Icon Design Tips:
- **Size**: 128x128 or 256x256 pixels (PNG format)
- **Style**: Match Minecraft's pixel art style
- **Colors**: Use vibrant colors that stand out
- **Theme**: Pickaxe, diamond, ore blocks, or your server logo

## Example Design Ideas:
1. **Diamond Pickaxe** - Classic VeinMiner symbol
2. **Ore Cluster** - Multiple colored ores together
3. **Your Server Logo** - KimDog Studios branding
4. **Explosive Effect** - Mining/blast effect

## Tools to Create Icons:
- **Piskel** (free, browser-based pixel art)
- **Aseprite** (paid, professional)
- **GIMP** (free, photo editor)
- **Paint.NET** (free, Windows)
- **Photoshop** (paid)
