# Changelog

All notable changes to KimDog SMP will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- **VeinMiner System**
  - Mine entire ore veins by holding Shift
  - Quest system with daily mining challenges
  - Mining statistics tracking
  - Upgrade system for enhanced mining
  - Configurable ore detection and limits

- **Enhanced Chat System**
  - Custom chat formatting
  - Automatic server announcements
  - Configurable message intervals
  - Admin controls for message management

- **Double Door Mechanics**
  - Automatic paired door opening/closing
  - Trapdoor support
  - Smart door pair detection
  - Synchronized operation

- **AntiCheat System**
  - Speed hack detection
  - Fly hack prevention
  - Reach hack monitoring
  - Configurable sensitivity
  - Real-time violation logging

- **Custom Commands**
  - `/fly` - Flight control
  - `/veinminer` - VeinMiner management
  - `/quest` - Quest system
  - `/chatmessages` - Chat controls
  - `/anticheat` - AntiCheat management
  - `/kimdogsmp` - Mod information and updates

- **Auto-Update System**
  - Automatic GitHub release checking
  - Player notifications on join
  - One-command download and installation
  - Progress tracking
  - Smart cleanup of old versions

- **Zoom Feature**
  - Configurable zoom keybind (default: Z)
  - Smooth zoom transitions
  - Adjustable zoom levels

- **Professional Initialization Banner**
  - ASCII art logo
  - Module loading progress
  - Detailed status messages
  - Load time tracking

### Configuration
- VeinMiner settings: `config/kimdog_smp/veinminer.json`
- Chat messages: `config/kimdog_smp/chatmessages.json`
- Double doors: `config/kimdog_smp/doubledoor.json`
- AntiCheat: `config/kimdog_smp/anticheat.json`
- Quest system: `config/kimdog_smp/quests/`

### Technical
- Built for Minecraft 1.21
- Fabric Loader required
- Fabric API dependency
- 6 core modules
- 15,000+ lines of code
- Comprehensive error handling
- Async operations for performance
